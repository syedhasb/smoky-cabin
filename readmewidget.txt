Read Me: Widget

This plugin allows for the widget to be added into the sidebar sections of the website. To add the widget into the site:

1.Go to appearance section and widget tab

2.You can see all the available sidebars for the widget to be implemented in

A new tab is created on the backend of wordpress. In this tab new posts can be created which will be displayed through the widget section. 
The widget will display posts from the widget section into the sidebar.
